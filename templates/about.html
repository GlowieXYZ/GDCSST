About

DCS Server Tracker (DCST) is a tool that allows you to keep track of DCS World online servers. It tracks the servers, the number of players and which scenarios are played on them and tries to present all the information gathered in something that is easy to see at a glance.

Features

This site features the following:

  • An up to to date list of DCS Servers (kindly provided by digitalcombatsimulator.com)
  • Sorting functions
  • Per server graph of recently active players along with additional information
  • Providing tools for the community like server info in JSON or a Prometheus exporter (See the Tech section)

Tech

Tools used

GDCSST is made using the following tools:

  • Atom Editor
  • Git
  • Go
  • Prometheus
  • Prometheus-PNG
  • Redis
  • AWS CloudFront (Which as a pain to setup the first time)

JSON data of servers

This site offers the ability to get server data as stored in Redis in JSON format. This is achieved by setting the "Accept" header to "application/json" as so:

  • curl -H "accept: application/json" https://dcs.glowie.xyz/servers/136.243.51.111:10300

This is setup to allow your script or discord-bot to collect metrics on your server without having to hammer the DCS servers and go through all the hoops. Hope it helps.

Prometheus Exporter

This service also provides a public Prometheus exporter for use. The URL is https://dcs.glowie.xyz/metrics. This allows you to collect your own data and create nice graphs than I ever can.
